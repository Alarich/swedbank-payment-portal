$cardHolderNameControl

$cardHolderNameControl : string

Field is optional and is used to control the display of the cardholder name field.

Populate with “show” otherwise Cardholder Name will not be visible on capture page. If value of show is supplied, cardholder name field will be displayed on hosted page. Any other value or omission of this field will not display the cardholder name field.

$goBackLink : string

Fully qualified URL for the Go Back link that is displayed on the capture page.

A secure (https) URL must be provided. If left blank, the function will not work.
