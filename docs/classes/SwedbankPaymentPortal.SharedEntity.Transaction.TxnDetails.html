$merchantReference

$merchantReference : string

Merchants can use a value such as an order number or an invoice number as the merchant reference. To allow cardholders to repeat a transaction that was declined and keep the same order number or invoice number, the merchantreference must be modified by appending extra characters for each subsequent attempt.
